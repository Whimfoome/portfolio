## Games

World of Monkeys (WIP)

World of Monkeys is a mobile game being made in Godot engine using C# as primary language. For 3D creation (characters, models, maps) Blender is used, for 2D GIMP and Krita.

It's a 3D Side-scrolling RPG game about Monkeys focused on platforming and exploring while encountering enemies and allies.

Some of the systems include:
- Custom Character Controller - a kinematic player controller for platforming, smooth movement (acceleration and deceleration), smooth camera follow, handles slopes, variable jump, coyote timer, jump buffer and more
- Dynamic Inventory System
- Quest System
- UI controls for comfortable mobile experience
- Combat and AI
- Save/Load, Checkpoints, Level Loader
